#  Fingerprint Recognition Web App

##  Overview
This project is a hybrid fingerprint recognition system combining:
- ORB feature matching (OpenCV)
- CNN-based similarity scoring
- Flask web interface

##  Features
- Upload two fingerprint images
- Compute similarity score
- ORB keypoint matching visualization
- Graphical comparison (bar chart)
- Final decision: Same / Different fingerprint

##  Methodology
1. Image preprocessing (CLAHE, resizing)
2. Feature extraction (ORB + CNN)
3. Cosine similarity calculation
4. Threshold-based classification

##  Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- Flask
- Matplotlib
